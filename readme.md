## Driving simulator Brecht Evens

Driving simulator accompanying the master dissertation of Brecht Evens, submitted to fulfil the graduation requirements for
the degree of "master in de wiskundige ingenieurstechnieken" at K.U. Leuven. 

### Project structure

* **customopengen-library** : Custom version of Opengen python package
* **driving-simulator** : Source code of the driving simulator
